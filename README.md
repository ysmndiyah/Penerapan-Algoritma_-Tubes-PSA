# 📖 Deskripsi Project

WarungPOS Algorithm Simulator adalah aplikasi berbasis web yang dibuat untuk mengimplementasikan beberapa strategi algoritma dalam simulasi sistem kasir modern.

Project ini menggabungkan:

* Greedy Algorithm
* Linear Search
* Bubble Sort

ke dalam satu ekosistem aplikasi bernama **WarungPOS**.

Selain implementasi algoritma, aplikasi juga memiliki tampilan modern dan interaktif untuk memvisualisasikan proses algoritma secara realtime.

---

# ✨ Fitur Utama

## 🟢 Greedy Algorithm

Simulasi optimasi pengembalian uang menggunakan strategi greedy.

### Fitur:

* Input total belanja
* Input uang bayar
* Perhitungan otomatis pecahan uang
* Visualisasi langkah greedy
* Kompleksitas algoritma
* Statistik hasil
* Riwayat simulasi

---

## 🔎 Linear Search

Implementasi pencarian produk menggunakan Linear Search.

### Fitur:

* Searching realtime
* Pencarian berdasarkan nama produk
* Menampilkan jumlah produk ditemukan
* Visualisasi hasil pencarian

---

## 🟣 Bubble Sort

Implementasi Bubble Sort untuk pengurutan produk berdasarkan harga.

### Fitur:

* Sorting harga produk
* Visualisasi hasil sorting
* Pengurutan ascending

---

# 🧠 Algoritma yang Digunakan

| Algoritma        | Implementasi            |
| ---------------- | ----------------------- |
| Greedy Algorithm | Optimasi kembalian uang |
| Linear Search    | Pencarian produk        |
| Bubble Sort      | Pengurutan harga produk |

# 📊 Kompleksitas Algoritma

| Algoritma     | Kompleksitas |
| ------------- | ------------ |
| Greedy        | O(n)         |
| Linear Search | O(n)         |
| Bubble Sort   | O(n²)        |
